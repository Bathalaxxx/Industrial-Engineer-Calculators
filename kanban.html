Kanban Methods and When to Use Them

Constant Demand, Constant Lead Time

Use when both demand and lead time are stable. Calculates the minimum number of Kanban cards needed. Formula: N = (D * L) / C

Where: D = Average demand, L = Average lead time, C = Container size

Variable Demand, Constant Lead Time

Use when demand fluctuates but lead time is fixed. Includes safety stock for demand variability. Formula: N = (D * L + Z * σ_d * √L) / C

Where: Z = Z-score for service level, σ_d = Standard deviation of demand

Constant Demand, Variable Lead Time

Use when demand is stable but lead time varies. Includes safety stock for lead time variability. Formula: N = (D * L + Z * D * σ_L) / C

Where: σ_L = Standard deviation of lead time
