Batch Sizing Methods and When to Use Them

Constant Demand, Constant Setup Time

Use when demand and setup times are stable. Equivalent to the EOQ model. Formula: Q = √(2DS/H)

Where: D = Annual demand, S = Setup cost, H = Holding cost

Variable Demand, Constant Setup Time

Use when demand fluctuates but setup time is fixed. Includes safety stock for demand variability. Formula: Q = √(2DS/H) + Z * σ_d

Where: σ_d = Standard deviation of demand, Z = Z-score for service level

Constant Demand, Variable Setup Time

Use when demand is stable but setup times vary. Adjusts batch size for setup time variability. Formula: Q = √(2DS/H) + Z * D * σ_s / P

Where: σ_s = Standard deviation of setup time, P = Production rate
